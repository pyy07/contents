# agent-tools

## ADDED Requirements

### Requirement: 工具/命令执行的访问与权限策略

系统 SHALL 防止工具或命令执行中的危险操作：**项目目录**（以项目根为基准的路径范围内）内可任意操作，无需额外授权；**非项目目录**（项目根之外的路径）的访问或写操作 MUST 先请求用户权限，在用户确认前不得执行。未获权限时，工具 SHALL 返回明确提示（如「需授权访问项目外路径」），不执行实际操作。

#### Scenario: 项目目录内操作直接执行

- **WHEN** Agent 调用的工具涉及路径（如 list_dir、read_file、write_file、run_code 的工作目录）且该路径在项目根下
- **THEN** 系统直接执行操作，不弹出权限请求

#### Scenario: 非项目目录操作需请求权限

- **WHEN** Agent 调用的工具涉及路径且该路径在项目根之外（或解析后指向项目外）
- **THEN** 系统不立即执行，而是发起权限请求（如提示用户确认）；仅在用户授权后执行，否则返回「需授权」类结果

#### Scenario: 未授权时不执行并返回明确提示

- **WHEN** 用户拒绝或未响应非项目目录的权限请求
- **THEN** 工具返回明确错误或提示信息（如「访问项目外路径需用户授权」），不执行任何写或读操作

### Requirement: 内置工具可供 Agent 调用

系统 SHALL 提供一组内置工具（Tools），供主 Agent 与 Sub-Agent（或按策略仅主 Agent）调用，用于完成基础能力。工具以统一接口暴露（name、description、params），与 Skill 类似但为系统内置、不归属单一领域 Agent。涉及路径的工具 SHALL 遵循「工具/命令执行的访问与权限策略」。

#### Scenario: Agent 查询可用内置工具列表

- **WHEN** 主 Agent 或已授权 Sub-Agent 请求可用内置工具列表
- **THEN** 系统返回工具名称、描述及参数 schema，供 Agent 选择并调用

#### Scenario: Agent 调用内置工具并得到结果

- **WHEN** Agent 传入工具名称与合法 params 调用某内置工具
- **THEN** 系统执行该工具并返回结构化结果（或错误信息），与任务结果契约一致

### Requirement: 执行代码工具

系统 SHALL 提供内置工具「执行代码」（如 run_code / execute_code），使 Agent 可编写并执行代码片段或脚本（如 Python），并在受控环境中获取执行输出或错误。若指定工作目录，项目目录内可直接执行，项目外 MUST 先请求用户权限，未授权则不执行。

#### Scenario: 执行代码成功返回输出

- **WHEN** Agent 调用执行代码工具，传入代码内容与可选工作目录或解释器（工作目录在项目内或已获授权的项目外路径）
- **THEN** 系统在受控环境中执行代码并返回 stdout、stderr 及退出状态，供 Agent 继续决策

#### Scenario: 执行代码失败返回错误

- **WHEN** 执行过程中发生异常或超时
- **THEN** 工具返回失败状态与可读错误信息，不崩溃主流程

#### Scenario: 工作目录在项目外且未授权时不执行

- **WHEN** Agent 指定工作目录为项目外路径且用户未授权
- **THEN** 工具不执行代码，返回需授权访问项目外路径的提示

### Requirement: 查看目录与文件列表工具

系统 SHALL 提供内置工具「列出目录/文件」（如 list_dir / list_directory），使 Agent 可查看指定路径下的目录与文件列表（名称、类型、可选元数据），用于探索工作区或素材结构。

#### Scenario: 列出目录内容成功

- **WHEN** Agent 调用列出目录工具并传入路径（项目根内则直接执行；项目外则需用户授权后执行）
- **THEN** 系统返回该路径下的条目列表（如名称、是否为目录），供 Agent 规划下一步操作

#### Scenario: 路径不存在或不可访问时返回错误

- **WHEN** 传入路径不存在、无权限访问、或为项目外路径且用户未授权
- **THEN** 工具返回明确错误信息，不暴露敏感路径细节

### Requirement: 读写文件工具（可选首版范围）

系统 SHALL 提供内置工具「读取文件」（read_file）与「写入文件」（write_file），使 Agent 可读取或写入指定路径文件内容。路径在项目目录内可直接操作；路径在项目外时 MUST 先请求用户权限，未授权则不执行。首版可进一步限制为仅允许项目根下或白名单路径。

#### Scenario: 读取文件成功返回内容

- **WHEN** Agent 调用读取文件工具并传入路径（项目内直接执行，项目外须已获用户授权）
- **THEN** 系统返回文件文本内容（及编码信息），供 Agent 分析或汇总

#### Scenario: 写入文件成功

- **WHEN** Agent 调用写入文件工具并传入路径与内容（项目内直接执行，项目外须已获用户授权）
- **THEN** 系统将内容写入该路径并返回成功状态，供 Agent 确认
